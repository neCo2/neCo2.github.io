Clip Studio Paint

I'm one of the 7 people who actually use CSP to scanlate. While its typesetting features leave a lot to desire, it's very attractive to redraw with the same tools most mangaka use to draw manga in the first place.

Unlike Photoshop, CSP has no support for scripting, so you either have to use very simple auto actions, or make do with external tools to help with your workflow. For example, there is no equivalent to TyperTools available for CSP, which is why I've painstakingly written myself something similar in AHK. It's a little janky, but it works most of the time, allows me to quickly paste text from scripts onto a page, and it can even select fonts and pre-format text.
On request I'll gladly send you the entire thing, but be aware that the code is largely uncommented and you'll have to adjust a lot of shit yourself. And I absolutely do not want to help you with any of that.

Auto Actions

While CSP's auto actions are very simple, they will still help you streamline your workflow. God knows there are many trivially simple things you'll have to repeat over and over again in this hobby. Here are some Auto Actions that make my scanlation life easier. Download the entire set here. Descriptions of the Actions can be found below.

Redraw Layers V3

Automatically create redrawing and text folders.

Bubble Select V3

First select bubbles with a magic wand tool with Area scaling set to a negative value (-3px works for most manga), then run this action to automatically fill the entire bubble with white.

Screentone

Manually select a repeatable pattern from a page and transform it into an image layer with a mask. You then have to manually enable tiling, and ジャジャ～ン, you've got a screentone layer.

Vector Stroke

Handwrite text on a vector layer. This action will then duplicate the layer, and change the line colour of the below layer to white. This allows you to then manually increase the brush size of this layer, creating a stroke effect.

convert to image layer

Self-explanatory. The noBack verison of the action does not preserve the original layer. Useful to move around small illustrations in bubbles or similar.

My Standard Workflow

  1. Whiteout
    2. Paint over all text that will be replaced with white/black.
  2. Redraw Sketch
    3. If complex redraws are required, I will first sketch them out on a low opacity layer.
  3. Redraw Colour/Gray/Screentones
  4. Redraw Lineart
  5. Typeset
    6. Paste lines from the script onto the page and try to make them look nice.
  6. Handwritten Text
    7. If text is handwritten in the raw, I will usually handwrite it too. Most people just use a messier font. I prefer handwriting as it allows me to more closely match the style of the original text. But this flexibility, unsurprisingly, comes at the cost of a whole lot of time and effort.
Some people prefer to typeset before redrawing to reduce the amount of redraws they have to do. That's most certainly the more sensible way to do things, but my pride/autism does not allow me to do that.
